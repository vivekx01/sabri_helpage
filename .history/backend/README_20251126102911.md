# Sabri Helpage - Backend

This is a minimal Express + MongoDB backend scaffold for the Sabri Helpage project.

Features included:
- Express server (`server.js`)
- MongoDB connection using `mongoose` with `.env` config
- Models: `Teacher`, `Animal`, `Contact`, `Donation`, `Story`, `Cause`, `Subscription`
- Controllers and routes following the models/controllers/routes pattern
- Seeder (`seed.js`) that inserts sample teachers and animals if collections are empty
- Routes mounted under `/apixyz` and `/api`:
  - `GET /apixyz/animals` - list animals
  - `GET /apixyz/teachers` - list teachers

New API endpoints (mounted under `/api`):
- `GET /api/causes` - list causes
- `POST /api/causes` - create a cause
- `GET /api/stories` - list stories
- `POST /api/stories` - create a story
- `POST /api/contact` - submit a contact message
- `GET /api/contacts` - list contact messages
- `POST /api/donate` - create a donation record
- `GET /api/donations` - list donations
- `POST /api/subscribe` - add a newsletter subscription
- `GET /api/subscriptions` - list subscriptions

Quick start
1. Copy `.env.example` to `.env` and set `DB_URI` (or `MONGO_URI`) to your MongoDB connection string.

2. Install dependencies (run in `backend`):

```powershell
cd backend
npm install
```

3. Seed database (optional):

```powershell
npm run seed
```

4. Start server:

```powershell
npm start
# or for development with nodemon
npm run dev
```

Server will run on `http://localhost:5000` by default.

Example curl commands:

```powershell
curl http://localhost:5000/api/causes
curl -X POST http://localhost:5000/api/contact -H "Content-Type: application/json" -d '{"name":"Test","message":"Hello"}'
curl -X POST http://localhost:5000/api/subscribe -H "Content-Type: application/json" -d '{"email":"you@example.com"}'
```
