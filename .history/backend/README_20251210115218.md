# Sabri Helpage Minimal Backend

## Features
- Node.js + Express + MongoDB
- Single flexible Page model (slug, title, content, sections)
- REST API: GET/PUT /api/content/page/:slug
- CORS enabled for frontend
- Minimal, efficient, easy to extend

## Setup
1. Install dependencies:
   ```
   npm install
   ```
2. Set up MongoDB and update `.env` if needed.
3. Start the server:
   ```
   npm run dev
   ```
4. Connect your frontend to `http://localhost:5000/api/content/page/{slug}`

## Example Page Document
```
{
  "slug": "donate",
  "title": "Donate",
  "content": "Support our mission!",
  "sections": [
    { "type": "hero", "content": { "headline": "Transform Lives" } },
    { "type": "form", "content": { "fields": ["amount", "name"] } }
  ]
}
```
