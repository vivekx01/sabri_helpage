# MongoDB Connection Fix

## Issue
- MongoDB connection failed: connect ECONNREFUSED ::1:27017, connect ECONNREFUSED 127.0.0.1:27017
- Local MongoDB not running or not installed

## Changes Made
- Removed deprecated `useNewUrlParser` and `useUnifiedTopology` options from `backend/config/db.js`

## Next Steps
1. Set up MongoDB Atlas (free tier):
   - Go to https://www.mongodb.com/atlas
   - Create a free account
   - Create a new cluster
   - Create a database user
   - Whitelist your IP address (or 0.0.0.0/0 for all)
   - Get the connection string from "Connect" > "Connect your application"

2. Update .env file:
   - Add `MONGO_URI=your_mongodb_atlas_connection_string`
   - Replace `<password>` with your database user password
   - Replace `<database>` with your desired database name (e.g., ngo-db)

3. Restart the backend server
